## Getting Started

### Prerequisites

- Node.js v22.14.0 or higher
- npm

### Installation

```bash
npm install
```

### Development Server

```bash
npm run start
```

Navigate to `http://localhost:4200/`

### Build

```bash
npm run build
```

Build artifacts will be stored in the `dist/` directory.
